Simple, but ejectable modern docs for simple devs.

Write docs in Markdown. Push to GitHub. Get a beautiful docs site.

## Quick Start

Create a `docs` folder in your repo:

```bash
mkdir -p docs
```

Add `gittydocs.jsonc`:

```jsonc
// docs/gittydocs.jsonc
{
  "site": {
    "name": "My Project",
    "repo": {
      "owner": "yourusername",
      "name": "yourrepo",
      "ref": "main",
      "docsPath": "docs",
    },
  },
}
```

Add a page (i.e. `docs/index.mdx`):

```mdx
---
title: Welcome
---

# Welcome

Your content here.
```

Push to GitHub. Pick a deployment option:

- [GitHub Pages](/deploy/github-pages)
- [Vercel](/deploy/vercel)
- [Cloudflare Pages](/deploy/cloudflare)
- [Netlify](/deploy/netlify)
